# Smart Navigation System

## Problem Description
This project finds a path between two nodes using search algorithms.

## Algorithms Used
- Breadth First Search (BFS)
- Depth First Search (DFS)

## Features
- BFS finds shortest path
- DFS explores depth

## Execution
Run main.py

## Sample Output
BFS Path: ['A', 'C', 'F']
DFS Path: ['A', 'B', 'E', 'F']
